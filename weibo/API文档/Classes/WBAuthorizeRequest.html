Inherits from WBBaseRequest : WBDataTransferObject : NSObject
Declared in WeiboSDK.h

Overview

第三方应用向微博客户端请求认证的消息结构

第三方应用向微博客户端申请认证时，需要调用 [WeiboSDK sendRequest:] 函数， 向微博客户端发送一个 WBAuthorizeRequest 的消息结构。 微博客户端处理完后会向第三方应用发送一个结构为 WBAuthorizeResponse 的处理结果。
